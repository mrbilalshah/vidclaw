# Awesome Lists to Submit VidClaw To

Submit VidClaw via pull request to each repo, following their contribution guidelines.

- **Category:** Automation / Software Development - Project Management
- **Stars:** 200k+
- **Notes:** Biggest self-hosted list. Submit under "Automation" or "Software Development - Project Management". Strict PR requirements — needs license badge, proper description format.

### Bonus: Create your own
- **awesome-openclaw** — Start an awesome list for the OpenClaw ecosystem. Seed it with VidClaw and other OpenClaw tools/skills. This becomes a long-term SEO asset.

## Submission Tips
1. Read each repo's CONTRIBUTING.md before submitting
2. Follow their exact format (description length, badge requirements, alphabetical order)
3. One PR per list — don't batch
4. Write a clear PR description explaining what VidClaw is and why it fits
5. Be patient — some lists take weeks to review PRs
