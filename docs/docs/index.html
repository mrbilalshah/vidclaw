Documentation

Get your self-hosted AI dashboard up and running in under a minute. This guide covers installation, configuration, and daily usage.

Prerequisites for Your Self-Hosted AI Dashboard

Before installing VidClaw, make sure you have:

  • Node.js 18+ installed on your server
  • A running OpenClaw instance
  • SSH access to your server (for the secure tunnel)

Quick Install — OpenClaw Agent Management Setup

Clone the repository

Clone VidClaw into your OpenClaw workspace:

cd ~/.openclaw/workspace
git clone https://github.com/madrzak/vidclaw.git dashboard

Run the setup script

The setup script installs dependencies, builds the frontend, and creates a systemd service:

cd dashboard
./setup.sh

This will install npm packages, build the Vite frontend, and register VidClaw as a system service.

Access via SSH tunnel

VidClaw binds to localhost:3333 for security. Access it from your local machine:

ssh -L 3333:localhost:3333 root@your-server

Then open http://localhost:3333 in your browser.

AI Cost Tracking Configuration

VidClaw automatically detects your OpenClaw usage data. The AI cost tracking panel reads from OpenClaw's usage logs and displays:

  • Real-time token counts per model
  • Estimated costs based on current API pricing
  • Rate limit progress bars matching Anthropic's windows

Tip: Switch models directly from the VidClaw navbar. Your agent will use the new model on its next session.

Managing Tasks with the Kanban Board

The Kanban board is VidClaw's primary interface for OpenClaw agent management. Tasks flow through four columns:

  1. Backlog — ideas and future work
  2. Todo — queued for your agent
  3. In Progress — your agent is currently working on these
  4. Done — completed tasks with timestamps

Drag cards between columns, assign priorities (P1–P4), and attach skills to tasks. Your agent picks up Todo items automatically based on priority.

Soul Editor — Customize Your Agent

The Soul Editor lets you shape your agent's personality and instructions. Edit the SOUL.md file through a visual interface with:

  • Live preview of changes
  • Version history with one-click revert
  • Six starter templates (professional, casual, technical, etc.)

Skills Manager

Skills extend your agent's capabilities. Through VidClaw's Skills Manager you can:

  • Browse all bundled and custom skills
  • Enable or disable skills with a toggle
  • Create new custom skills with the built-in editor

Changes take effect on your agent's next session — no restart required.

Updating VidClaw

cd ~/.openclaw/workspace/dashboard
./update.sh

The update script pulls the latest changes, rebuilds the frontend, and restarts the service.

Uninstalling

cd ~/.openclaw/workspace/dashboard
./uninstall.sh

This stops the service, removes the systemd unit, and cleans up. Your OpenClaw data is never touched.
